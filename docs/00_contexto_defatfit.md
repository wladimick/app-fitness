# DeFatFit — Contexto del proyecto

App personal de entrenamiento, planificación y seguimiento físico.
Stack: HTML, CSS, JavaScript vanilla con Supabase como backend.
Mobile-first, sin frameworks, deploy en Netlify.

Módulos incluidos: calendario, rutina del día, perfil físico Body Pro, recomendaciones por músculo, administración.
Módulos excluidos: nutrición avanzada, tienda, pagos, comunidad, chat, IA avanzada.
